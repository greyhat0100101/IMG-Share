# IMG-Share

App simple: **Organizaciones → Álbums → Imágenes** (Cloudinary) + cache básico (Redis).

## Requisitos
- Node.js 18+
- npm

## Instalar
```bash
npm install express mongoose dotenv multer cloudinary redis cors streamifier
npm install -D nodemon
```

## Ejecutar
```bash
npm start
# o en dev:
npm run dev
```

Abrir: http://localhost:3000
